Dark-themed CV site scaffold using Next.js, Tailwind CSS and react-three-fiber.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run in dev:

```bash
npm run dev
```

3. Deploy to Vercel: push to a Git provider and import the repo in Vercel (Next.js is supported out-of-the-box).

Notes:
- The 3D project carousel is implemented with react-three-fiber and loads remote images (Unsplash placeholders). Replace images and project links in `pages/index.js`.
- Language: JavaScript. Theme: dark.
